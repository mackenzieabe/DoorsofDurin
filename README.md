# Doors of Durin-Password Generator 

## How It Works
Our of password ideas? Doors of Durin can fix that for you. When the 'Generate Password' button is clicked the user is presented with a series of questions/prompts to create a random password. The user is guided in picking the length of the password and the characters used. Once the user answers all of the prompts they are given a random generated password.
